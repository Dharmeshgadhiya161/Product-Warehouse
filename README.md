# 📱 B2B Ordering App for Dealers & Suppliers

### Billing, Tracking & Inventory System (Android)

🚀 A scalable Android application built to streamline order management between dealers and suppliers.

---

## 📌 Project Overview

This application enables dealers to browse product catalogs, search and filter items, and place orders efficiently.
Suppliers can manage orders, track transactions, and generate billing files (CSV & PDF) for seamless processing.

Designed for small businesses, wholesalers, and distributors to digitize and optimize their operations.

---

## ✨ Key Features

* 🛒 Category-wise product browsing with search & filters
* 📦 Real-time order placement and tracking
* 🧾 GST-compliant invoicing (PDF & CSV export)
* 🚚 Delivery challan generation
* 📊 Inventory and order management
* 🔔 Push notifications for order updates
* 📶 Offline functionality support

---

## 🛠 Tech Stack

* **Language:** Kotlin, Java
* **Architecture:** MVC
